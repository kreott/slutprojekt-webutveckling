# Assembly Playground
 
A javascript-based Assembly emulator.
 
## Controls
 
- ``RUN`` run the full program (instant or with a step delay via the interval slider)
- ``STEP`` ‚Äî execute one instruction at a time
- ``STOP`` ‚Äî reset everything
 
## Instructions
 
- ``MOV`` dest, src
- ``ADD`` dest, src
- ``SUB`` dest, src
- ``MUL`` dest, src
- ``CMP`` dest, src
- ``JMP`` label
- ``JE`` label
- ``JNE`` label
 
Comments start with `;`.
